# DocMatch Automator

An automated document matching and reconciliation system for small businesses. Upload, match, and reconcile purchase orders, invoices, and delivery receipts automatically.

## Features

- Bulk document upload
- Automatic data extraction
- Document matching
- Mismatch flagging
- Reconciliation reports

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Technologies Used

- React
- Material-UI
- React Dropzone
- PDF.js
- Chart.js
